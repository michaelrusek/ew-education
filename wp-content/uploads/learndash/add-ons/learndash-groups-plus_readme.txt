LearnDash Groups Plus - The Groups Plus add-on empowers you to sell B2B learning solutions with powerful group management and reporting tools.

== Description ==

Extend the functionality of LearnDash Groups to allow your users to create and control their own organization. Users purchasing an organization can purchase a configurable number of seats to control the number of team members allowed within the organization.

Front end management, reporting and grading means that your users can easily control the organization without needing access to the WP administrator dashboard.

== Installation ==

If the auto-update is not working, you always have the option to update manually. Please note, a full backup of your site is always recommended prior to updating. 

1. Deactivate and delete your current version of LearnDash Groups Plus.
2. Download the latest version of LearnDash Groups Plus from your account portal - https://account.learndash.com
3. Upload the zipped file via PLUGINS > ADD NEW, or to wp-content/plugins.
4. Activate the LearnDash Groups Plus plugin via the PLUGINS menu.
